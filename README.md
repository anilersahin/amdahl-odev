# amdahl-odev
# Amdahl Yasası Ödevi

Bu ödev, paralel programlama dersinde anlatılan **Amdahl Yasası** konusunu
daha iyi anlamak amacıyla Python dili kullanılarak yapılmıştır.

## Açıklama
Programda, bir işlemin paralel yapılabilen kısmı ile seri kalan kısmının,
işlemci sayısı arttıkça toplam çalışma süresini ve hızlanmayı nasıl
etkilediği hesaplanmaktadır.

Kullanıcıdan paralelleştirilebilir oran ve işlemci sayısı alınarak
Amdahl Yasası formülü uygulanmıştır.
